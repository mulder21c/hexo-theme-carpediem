## figcaption
generate figcaption element <br>
It must be used in `{% figure %}`

**Kind**: global function  
**Access**: public  
**Usage**  
```markdown
{% figcaption [caption string] %}
```
**Example**  
```markdown
{% figure %}
  <img src="xxxx.jpg" alt="" >
  {% figcaption "이미지 캡션" %}
{% endfigure %}
```

## figure
generate figure element

**Kind**: global function  
**Access**: public  
**Usage**  
```markdown
{% figure [attributes] [class] %}
```
**Example**  
```markdown
{% figure %}
  <img src="xxxx.jpg" alt="" >
{% endfigure %}
```
